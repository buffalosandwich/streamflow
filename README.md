# Streamflow Dashboard

A mobile-friendly web app to view real-time stream data (gauge height, water temperature, and flow) for 11 Colorado stations using the DWR API.

## Live Site
https://streamflow-fwrh.onrender.com

## How to Use
- Click the "Refresh Data" button to load the latest stream data.
- View current values and 10-day history graphs for each station.

## Deployment
This site is deployed using Render.com as a static site.
